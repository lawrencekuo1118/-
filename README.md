# Group 6 analytics coursework

This repository contains R and R Markdown coursework for the 2019 predictive
business analytics course, including weekly exercises, a Brazilian e-commerce
mapping midterm, and a Ta Feng retail analytics final project.

## Page Media Extractor

`media-extractor/` contains a modern Chrome/Edge extension that upgrades the
shared image-title/media downloader prototype. See
[`media-extractor/README.md`](media-extractor/README.md) for installation,
behavior, and limitations.
